== Leírás ==

A RendanIT SEO egy könnyű, gyors SEO plugin amely tartalmazza az összes fontos SEO funkciót:

**Főbb funkciók:**

* **Meta Title & Description** - Egyedi title és meta description minden oldalhoz
* **Polylang integráció** - Nyelvenként külön főoldal SEO, hreflang tagek automatikusan
* **Schema.org Structured Data** - LocalBusiness, Service, FAQ, Article, Breadcrumb schemák
* **Open Graph & Twitter Card** - Facebook/LinkedIn/Twitter megosztás optimalizálás
* **XML Sitemap** - Automatikus sitemap generálás hreflang támogatással
* **SEO Audit** - Oldal szintű és globális SEO elemzés
* **Google Preview** - Előnézet a Google találatokhoz
* **Fókusz kulcsszó elemzés** - Kulcsszó sűrűség, title/desc/URL ellenőrzés
* **Indexelés beállítások** - Noindex archívum, tag, szerző oldalakhoz
* **GTM & GA4 integráció** - Tracking kódok egyszerű beillesztése
* **SEO oszlop az admin listákban** - Gyors áttekintés az oldalak SEO állapotáról

**Polylang kompatibilitás:**
- Nyelvenként külön főoldal title és description
- Automatikus hreflang tagek minden fordított oldalhoz
- x-default hreflang az alapértelmezett nyelvhez
- Sitemap hreflang linkek
- Audit: hiányzó fordítások jelzése

== Telepítés ==

1. Töltsd fel a `rendanit-seo` mappát a `/wp-content/plugins/` könyvtárba
2. Aktiváld a plugint a 'Bővítmények' menüben
3. Menj a 'RendanIT SEO' menüpontra a beállításokhoz
4. Töltsd ki a Schema.org adatokat (Vállalkozás neve, cím, telefon, stb.)
5. Állítsd be a főoldal SEO title-t és description-t minden nyelvhez
6. Mentsd el a permalinkeket (Beállítások > Állandó hivatkozások > Mentés) a sitemap működéséhez
